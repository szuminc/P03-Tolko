After downloading

  1. Move both PNGs into P03-Tolko-Workout-PWA/icons/, replacing the old files.
  2. Commit: git add icons/ && git commit -m "New app icon"
  3. Push to main — the workflow will deploy, or push directly to gh-pages.
  4. On iPhone: long-press the home screen icon → Remove App, then re-add via Safari to get the new icon.
